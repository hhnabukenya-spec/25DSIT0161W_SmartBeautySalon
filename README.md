## Project Description
The Smart Beauty Salon Appointment Booking System is a web-based application that allows customers to view salon services and book appointments online. 
The system stores booking information in a database so the salon administrator can manage appointments efficiently.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## How to Run the Project
1. Install XAMPP.
2. Start Apache and MySQL.
   
3.Copy the project folder into:
c:\xampp\htdocs
open your browser and go to:
http://localhost/smart-beauty-salon

## Database Setup
1. Open phpMyAdmin.
2. Create a new database.
3. Import the file database.sql
4. Run the project.

## System Access
User Page:
http://localhost/smart-beauty-salon

Admin Page:
